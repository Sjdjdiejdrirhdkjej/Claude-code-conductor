# Tech Stack: Claude Conductor

## Core Runtime
-   **Language:** TypeScript
-   **Runtime:** Node.js
-   **Package Manager:** npm

## CLI Framework
-   **Library:** Commander.js or Yargs (Standard Node.js CLI libraries)
-   **Interactivity:** Inquirer.js or Prompts (for interactive menus)
-   **Styling:** Chalk (for terminal coloring), Ora (for spinners)

## Integration
-   **AI Engine:** Claude Code (via `claude` CLI execution)
-   **Context Storage:** Markdown (`.md`) and JSON (`.json`) files
